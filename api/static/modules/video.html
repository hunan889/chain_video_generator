参数说明
- 音色种子: 相同数字 = 相同音色，换一个数字就是另一个人的声音
- 语速: 0=非常慢 → 5=正常 → 9=非常快
- 口语化: 0=正式播报风格 → 9=口语随意、带语气词(嗯、啊、呢)
- 笑声: 0=不笑 → 2=自动在合适位置插入笑声
- 停顿: 0=语速连贯 → 7=频繁停顿/喘息(含 [uv_break] 呼吸声)
- 温度: 越低越稳定可预测，越高越随机自然
NSFW 场景建议: 口语化=7~9 + 停顿=5~7 + 笑声=1~2，模型会自动插入呼吸声([uv_break])和喘息效果
注: ChatTTS 的情感效果由模型自动控制插入位置，不支持手动在文字中插入标签。音色效果因种子而异，建议多试几个种子。
